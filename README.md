# Travel Booking Backend

Backend service for a Travel Booking platform built with **NestJS** and **MongoDB**.

This project is designed as a **production-ready modular monolith**, following real-world backend architecture and best practices.

---

## Tech Stack

- NestJS (TypeScript)
- MongoDB (Mongoose)
- JWT Authentication
- Swagger (OpenAPI)

---

## Getting Started

### Prerequisites

- Node.js (>= 18)
- MongoDB (local or MongoDB Atlas)

---

```bash
npm install
